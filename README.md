# Static Website Hosting using Cloud Storage + CDN

## Overview
This project demonstrates hosting a static website using cloud object storage and a CDN for global content delivery.

## Technologies Used
- HTML, CSS
- Git & GitHub
- AWS S3 (Static Website Hosting)
- AWS CloudFront (CDN)

## Architecture
User → CDN → Cloud Storage (Static Files)

## Steps Performed
1. Created static HTML/CSS website
2. Uploaded files to cloud storage
3. Enabled static website hosting
4. Configured CDN for global access
5. Stored source code in GitHub

## Outcome
- Publicly accessible website
- Faster global access via CDN
